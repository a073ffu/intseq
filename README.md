# intseq
Handle integer sequences

## How to build

1. [Install rye](https://rye-up.com/guide/installation/)

    ```bash
    curl -sSf https://rye-up.com/get | bash
    echo 'source "$HOME/.rye/shims"' >> ~/.profile
    ```

2. Sync all

    ```bash
    rye sync
    ```

## How to test

1. Start rye shell

    ```bash
    rye shell
    ```

2. Run pytest

    ```bash
    pytest
    ```
